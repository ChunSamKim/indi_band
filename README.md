    MYSQL - HEIDSQL
  #### 프론트엔드

  #### 백엔드

### 4. 구현 내용
#### 그룹 기능
  1. 그룹 CRUD 등 그룹 관리
  2. 그룹 내부 일정 설정 기능
  3. 그룹 내 게시글 CRUD 기능
  4. 해시태그로 그룹 검색 기능
#### 알림 기능
  1. 그룹 초대, 가입 요청 등 알림 전송
#### 채팅 기능
  1. 채팅방 생성, 초대, 채팅방 나가기 등
  2. 채팅이 오면 푸시 알림 기능
#### 기타 기능
  1. jwt를 활용한 토큰 기반 인증
